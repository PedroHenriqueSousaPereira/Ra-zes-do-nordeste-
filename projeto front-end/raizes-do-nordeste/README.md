# Raizes do Nordeste

Projeto da faculdade Uninter de front-end proposta do projeto é criar um site para uma rede de lanchonetes nordestina.

## Objetivo

Desenvolver um site com HTML, CSS e JavaScript, simulando:

- login e perfil;
- escolha de unidade;
- consulta de cardápio;
- detalhe dos produtos;
- carrinho;
- checkout;
- acompanhamento de pedido;
- fidelidade;
- perfil do usuário;
- dashboard do gerente;
- relatórios;
- estoque.

## Programas usados

- HTML5
- CSS3
- JavaScript
- LocalStorage para simulação de carrinho e pedido

## Como abrir

Abra o arquivo `index.html` no navegador.
